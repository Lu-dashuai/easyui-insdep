# easyui-insdep
JQuery EasyUI Of Insdep theme
------
非官方
官方地址: https://www.insdep.com/

# 使用
## maven
1. 添加仓库地址
```xml
<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
</repositories>
```

2. 添加依赖
```xml
<dependency>
	  <groupId>cn.javaer</groupId>
	  <artifactId>easyui-insdep</artifactId>
	  <version>1.5.1</version>
</dependency>
```
